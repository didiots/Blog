# ✈️ Travel Blog (Gruppenprojekt)

Ein responsives Travel-Blog-Template, entwickelt als Gruppenprojekt im Rahmen der **Developer Akademie**. 

---

## 🎨 Über das Projekt

Ziel des Projekts war es, eine mehrseitige und moderne Reise-Website mit verschiedenen Layouts (Artikel, FAQs, Infografiken) umzusetzen.

- 📐 **Layout:** Rein mit **Flexbox** aufgebaut
- 📱 **Responsive:** Optimiert für verschiedene Bildschirmgrößen
- ⚙️ **Tech-Stack:** Reines **HTML5** & **CSS3** (ohne JavaScript)

---

## 👥 Gruppenarbeit

Dieses Projekt wurde gemeinsam im Team umgesetzt. 

* **HTML5** – Semantischer Code & Struktur
* **CSS3** – Custom Styling, Flexbox-Layouts & Media Queries
